# VectorDrift
A retro-inspired arcade space shooter built in GameMaker, featuring momentum-based movement and fast-paced survival gameplay.

## Features
- **Feature A**: Player ship with momentum-based controls and basic weapon system

## Project Structure
- `/assets/` - Game sprites, sounds, and other assets
- `/scripts/` - Game scripts and logic  
- `/objects/` - Game objects
- `/rooms/` - Game levels/rooms

## Getting Started
This project requires GameMaker Studio. See [FEATURE_A.md](FEATURE_A.md) for details on the player ship implementation.
