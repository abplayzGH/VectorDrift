# Assets Directory

This directory contains game assets for VectorDrift.

## Structure
- `/sprites` - Game sprites and visual assets
- `/sounds` - Sound effects and music (future)
- `/fonts` - Game fonts (future)

## Placeholder Assets
For Feature A, basic geometric shapes can be used:
- Player ship: Triangle or arrow shape pointing up
- Laser: Small rectangle or line

## GameMaker Integration
These assets should be imported into GameMaker Studio and assigned to the corresponding objects.
